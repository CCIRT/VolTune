# vivado

## What is this directory?

This directory contains Vivado designs and projects.

## Directories

```
├── design-example: Design example which received from AIST and modified design
├── evm           : Test design for Cora-Z7 with UCD9224EVM evaluation board.
├── power         : Design for BER, Latency and Power measurement
├── vitis_src     : Source files for Microblaze
└── voltage       : Design for voltage setting time measurement
```
