# Removed Support Files

This directory originally contained support HDL associated with the 5 Gbps loopback transceiver design example.

These files were removed from the public repository because they are generated or provided as part of the Xilinx 7 Series FPGAs Transceivers Wizard example/support flow and may be subject to AMD/Xilinx licensing terms that are separate from the repository-wide MIT License.

Removed files:

- `txrx5g_support.v`
- `txrx5g_gt_usrclk_source.v`
- `txrx5g_clock_module.v`
- `txrx5g_common.v`
- `txrx5g_common_reset.v`

Users who already hold a valid Vivado license and need these support files for this design example may contact **Akram BEN AHMED**.

Any provision, regeneration, access, or use of these files remains subject to the user's own AMD/Xilinx Vivado license terms, including the applicable End User License Agreement.
