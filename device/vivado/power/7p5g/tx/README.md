# TX Design, 7.5 Gbps

This directory contains the TX-side design with transceivers set to 7.5 Gbps.

This design is used for BER, latency, and power measurement in the VolTune power-oriented transceiver case study.

## Overview

This is the TX-side design for the dual-board test flow at 7.5 Gbps.

## Details

The structure and usage of this design follow the same pattern as the 10 Gbps TX design.

Please refer to [`../../10g/tx/README.md`](../../10g/tx/README.md) for the detailed design description, test flow, register map, LED behavior, and typical error codes.
