# Documentation Assets

This directory contains supporting documentation and documentation-related assets for the VolTune repository.

## Contents

- [`PMBUS.md`](PMBUS.md), repository-specific notes on PMBus concepts, command usage, and data formats relevant to VolTune
- [`VolTune_arXiv_2026.pdf`](VolTune_arXiv_2026.pdf), arXiv companion paper PDF included in the repository
- [`design_docs/`](design_docs/), documentation images and related assets reused by the repository README files

## Notes

- The files under [`design_docs/img/`](design_docs/img/) are used by several README files across the repository.
